규칙:
- 각 종속항 보고서는 반드시 `===청구항 N===` 줄로 시작하세요. N은 입력에 주어진 청구항 번호 그대로 사용합니다.
- 입력으로 주어진 종속항을 하나도 빠짐없이 모두 작성하세요.
- 각 청구항 본문은 `### [추가 구성]` 다음 줄의 `(A) ...` 판정 형식부터 바로 시작합니다. Phase 1 대제목과 Phase 2는 출력하지 않습니다.

[분석 가이드라인]
1. 부모 청구항의 구성은 이미 검토되었으면 별도 분석을 생략합니다. 부모항이 입력 또는 이전 컨텍스트에 없으면 그 실체를 추정하지 말고, '에 있어서' 뒤에 직접 기재된 추가 기술 특징만 분석합니다.
2. 해당 종속항만의 추가 구성(부모항에 없는 특징)에 집중하여 담당 인용발명과 대비합니다.
3. 각 추가 구성에 대해 대응 여부와 구체적 근거를 서술합니다.
4. 대응된다고 판단하면 그 근거를, 대응되지 않는다고 판단하면 유사한 점과 불일치 이유를 명시합니다.
5. 부모항이 확인되는 종속항만 부모항의 인용발명 조합을 상속합니다. 부모항이 확인되지 않으면 현재 종속항의 추가 기술 특징에 대응하는 문헌만 사용합니다. 새로 결합할 수 있는 인용발명은 [인용 체인 정보]에 표시된 최대 1개뿐입니다.
6. 각 종속항에서 새로 추가된 인용발명은 해당 종속항 추가 구성에 대한 실제 거절 조합 문헌입니다. 인용발명 번호가 3 이상이어도 주지관용 입증자료나 보조자료로 축소하지 말고, 추가 구성의 직접 대응 근거와 결합 판단에 사용합니다.
7. 종속항 단계가 깊어질수록 부모항의 최종 결합 발명을 계속 상속하고, 현재 종속항에서는 [인용 체인 정보]에 표시된 새 인용발명 1개만 추가합니다.
8. 여러 발췌가 있는 경우 각 발췌의 출처와 현재 종속항의 추가 기술 특징과의 대응 관계를 구분하여 인용합니다.
9. `단일 추가 문헌 커버 상태`가 `대응 불충분`이면 두 번째 새 인용발명으로 보완하지 말고, 허용된 조합만으로 거절 근거를 완성하기 어렵다고 명시합니다.
10. 발췌는 각 종속항의 [담당 인용발명 대응 구성 데이터]의 quote와 위치를 근거로 사용합니다. 한국어 quote는 그대로 옮기고, 외국어(일본어·중국어·영어 등) quote는 인용 규칙에 따라 한국어 직역 또는 준직역 문장 다음 줄에 원문 quote와 위치를 괄호로 병기하십시오. 요약·의역은 발췌 뒤 별도 문장으로 쓰고, 데이터에 없는 원문은 만들지 않습니다.
